# Core Utilities

## 목적

애플리케이션 전역에서 사용되는 공통 클래스와 유틸리티를 관리한다.

## 주요 파일

| 파일명 | 설명 |
| :--- | :--- |
| `Resp.java` | 공통 응답 구조 (status, msg, body)를 위한 래퍼 클래스 |

## AI 작업 지침

- **일관성**: API 응답 시 반드시 `Resp.ok()` 또는 `Resp.fail()`을 사용하여 응답 형식을 통일한다.
- **HTTP 상태 코드**: 적절한 `HttpStatus`를 함께 반환하여 클라이언트가 에러를 식별할 수 있게 한다.
