# Database Configuration

## 목적

H2 인메모리 DB 초기화 및 테스트 데이터 설정을 관리한다.

## 주요 파일

| 파일명 | 설명 |
| :--- | :--- |
| `data.sql` | 초기 데이터 삽입 스크립트 (User, Board, Reply 등) |

## AI 작업 지침

- **데이터 무결성**: 연관 관계가 있는 테이블(`board_tb`, `reply_tb`)의 데이터를 삽입할 때 외래 키 참조 순서를 준수한다.
- **테스트 편의성**: 개발 및 테스트 단계에서 즉시 확인할 수 있는 충분한 샘플 데이터를 유지한다.
